# hareemm.github.io

## Cursor Work Session prototype (demo)

The collaboration prototype lives in **`work-session-prototype/`**.

```bash
cd work-session-prototype
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

If that folder is missing, you are on the wrong branch:

```bash
git fetch origin
git checkout cursor/work-session-collab-prototype-b5ca
```

Docs: `work-session-prototype/DEMO_SCRIPT.md` · `work-session-prototype/PRODUCT_NOTES.md`

**Note:** The static AIM site at the repo root (`index.html`) is not the Work Session prototype.
